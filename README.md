Production
==========
管理画面上に「制作会社へのお問い合わせフォーム」をで表示するためのプラグインです。
簡易的な機能しか備えていませんが、MITライセンスで公開していますので、安心してカスタマイズして頂けます。


Usage
-------
プラグイン管理画面のプラグイン一覧から、本プラグインの設定画面へ入ります。
メールアドレスと制作会社名を登録してください。


System Requirements
-------
baserCMS 3.1.0〜
